- ステージされている変更からコミットメッセージの候補を3つ提案してください
- 指示
  - `git diff --staged` でステージされている変更内容を確認する
  - `git log --oneline --no-merges -10` で直近のコミットメッセージのスタイルを確認する
  - 変更内容を分析し、簡潔な日本語一行のコミットメッセージ候補を3つ提案する
- 出力形式
  - 候補のみを簡潔に出力する（説明不要）
- 注意事項
  - ステージされている変更がない場合は、その旨を伝える
  - 複数の異なる目的の変更が含まれている場合は、コミットを分けることを提案する
